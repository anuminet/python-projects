## Projects

### Budget Tracker
A command-line personal finance tracker that saves your data to a CSV file, so your transactions persist between sessions. Features include:
- Add income and expense entries with a category and description
- View a full transaction history with a running balance summary
- View spending broken down by category

**Concepts used:** CSV file I/O, dictionaries, functions, input validation, `datetime`

---

### Password Generator
A customizable password generator that lets you choose:
- Length between 8 and 64 characters
- Whether to include uppercase letters, numbers, and/or symbols

Generates a new password instantly and lets you make another one without restarting.

**Concepts used:** `random`, `string`, functions, input validation loops

---

### To-Do List Manager
A simple task manager that saves your to-do list to a text file so it persists between sessions. You can:
- View all current tasks (numbered)
- Add new tasks
- Remove tasks by number

**Concepts used:** File I/O, lists, functions, input validation
